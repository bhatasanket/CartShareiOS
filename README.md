# App Design

The Design is very simple.

1. User can signup
2. User can sign in
3. User can create a family/group
4. User can invite others based on the userid/email to the group
5. When user signs in he/she should receive invites if any pending
6. User can create carts/toDo list which are part of family.
7. User can delete the cart if required
8. User can add items to the list(e.g. bread, milk…)
9. User can see who has added the item by clicking the item
10. User can checkoff an item once it is done
11. User can uncheck the item if needed
12. User can delete the item if needed
13. And some more...
